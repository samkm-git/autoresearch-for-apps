# Legacy IVRS PHP Source

This directory is a placeholder for the original PHP source code of the Hospital at Hand IVRS system.

## Historical Context
Originally located in the `old-code` root folder, this source code served as the input for the **Discovery Agent** during the modernization process. This logic was successfully extracted into [legacy_flow.json](../autoresearch/legacy_flow.json).

## Note on Content
This folder currently contains no files as the original source was tracked as an external submodule that has been detached. If you have the original PHP files (`ivrs.php`, `ivrs-handler.php`, etc.), they should be placed here to complete the case study documentation.
